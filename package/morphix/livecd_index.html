Welcome to OpenXPKI Live (beta)!

Thanks for trying OpenXPKI. You can start using the webinterface now.

Here are some hints as to what you might want to try first:

  • Login as CA Operator (using the 'Operator' method, with the username 'caop' and the password 'caop') and unlock the CA key from the login link on the index page. Just enter '1' in the part field and '1234567890', which is the example CA key passphrase for the live CD setup, in the passphrase field.
  • Logout, login again as a user (using the 'User' method, with the username 'user@example.com' and the password 'password') and create a new certificate request. As we are using Firefox, SPKAC is a reasonable choice for a first start. Experiment with the different settings if you like to see what OpenXPKI can offer.
  • Login as a RA Operator ('Operator' method, username and password 'raop') and see the certificate request created by the user. Note that there is a link to the RT ticket system. Login using "root" and "password" if you want to see the ticket. Approve it to issue the certificate.
  • Login as the user again and notice that you can download the certificate right from the index page. Install it in Firefox to see that this integration works.

If you want to use OpenXPKI Live to create a persistent CA on a storage medium (for example on a USB flash drive), read this documentation.

Enjoy the OpenXPKI test drive and be sure to report any bugs you might encounter. Remember that this is not an official release yet, so be prepared to live on the edge.
